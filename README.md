# 1FirstPass
Key Features

1. The master password to 1FirstPass are not stored on our server but are encrypted to private and public keys pair, which are used to identify the correct password when loggin in.

2. We do not store any of user's web/app secured passwords or credentials on our server. The secured passwords are stored in user's web browsers and on his/her local computer file system as a backup.

3. In case you want to store your old passwords generated by yourself, you can do so.

4. On any pc or laptop, www.1firstpass.com works on all major browsers - Edge, Chrome, Firefox, Safari.

5. It allows upload of the local backup file on to database of the browser, so all user's website credentials can be available.

6. 1FirstPass now also works on android devices(phone, tablet) and ios devices (IPhone, IPad, macBook). User have to send the backup file from one device to another, e.g. through email and download the file. Then 1FirstPass will upload the file to browser database and list all websites credential automatically.
